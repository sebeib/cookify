# cookify frontend

React-Frontend mit TypeScript, React Router und Mantine.

## Lokale Entwicklung

Installation:

```bash
npm install
```

Start:

```bash
npm run dev
```

Der Vite-Dev-Server laeuft standardmaessig auf `http://localhost:5173` und proxyt `/api` automatisch auf das Quarkus-Backend unter `http://localhost:8080`.
